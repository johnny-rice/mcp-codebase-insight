## Quick Start

1. **Installation**
   ```bash
   pip install mcp-codebase-insight
   ```

2. **Basic Usage**
   ```python
   from mcp_codebase_insight import CodebaseAnalyzer
   
   analyzer = CodebaseAnalyzer()
   results = analyzer.analyze_code("path/to/code")
   ```

3. **Running Tests**
   ```bash
   pytest tests/
   ```
